Всем привет, в этом видео мы начнем рассматривать простейшие типы данных в python:
-булевы значения, они могут быть равны True  или False
-целые числа 
-числа с плавающей точкой - числа с десятичной точкой 3.14159 и экспоненты 1.0e8

Начнем с булевых значений. В питоне объекты булева(логического) типа данных моргут иметь только два значения
True или False. Иногда мы будем использовать их напрямую, а иногда оценивать истинность других типов по их значениям.
Специальная функция bool() может преоброзовать люой тип данных в python в булев.

Для не нулевых чисел эта функция вернет True:
print(bool(True))
print(bool(1))
print(bool(37))
print(bool(-37))

Нулевые значения преобразуютя в значение False
print(bool(False))
print(bool(0))
print(bool(0.0))

Это все что нужно знать про тип данных bool


Целые числа - это числа без дробей и десятичной точки.

Если число не 0 оно не может начинаться с 0 так как Python будет гененрировать исключение
print(01)

При записи целого нельзя пользоваться запятыми

a = 1,000,000
print(a)
print(type(a))

Вместо числа миллион мы лочим кортеж с тремя значеними, однако в качестве разделителя мы можемм использовавть прочерк

a = 1_000_000
print(a)
print(type(a))

Операции с целыми числами

+   сложение
-   вычетание 
*   умножение 
/   деление с плавающей точкой
//  целочисленое деление
%   остаток от деления
** возведение в степень

divmod(9, 5)

Приоритет операций в питоне такой же как и в математике но 
a = - 3 ** 2 равнозначно a = - (3 ** 2), поэтому для получения возведение -3 во 2ую степень
нам надо -3 взять в скобочки a = (-3) ** 2

Что бы превети тип данных на целочисленный используется функция int()
int(True)
int(False)

При преоброзовании чичла с плавающей точкой отсекается все, что находится полсле точки.
int(98.6)
int(1.0e4)

из текстовой строки

int('99')
int('-23')
int('+12')
int('1_000')

В Python3 переменная инт может быть любого размера

google = 10**100
print(google)
print(google * google)

Числа с плавающей точкой (в Python они называются float) имеют десятичную точку.
print(5.)
print(5.0)
print(05.0)

Числа с плавающей точкой могут содержать десятичную целочисленную экспоненту после буквы e:

print(5e0)
print(5e1)
print(5.0e1)
print( 5.0 * (10 ** 1))

Для разделения цифр можно использовать символ «нижнее подчеркивание» (_), как и в случае с целыми числами:

print(1_000_000.0)
print(1.001)

Числа с плавающей точкой обрабатываются так же, как и целые: вы можете использовать
операторы +, −, *, /, //, **, % и функцию divmod()

Для того чтобы преобразовать другие типы в тип float, следует использовать
функцию float(). Булевы значения по-прежнему обрабатываются как небольшие
числа:

print(float(True))
print(float(False))

При преобразовании целого числа типа int в число типа float просто добавляется десятичная точка:
print(float(98))

Можно конвертировать и строку с символами, представляющими собой корректные числа с плавающей точкой,
 — цифры, знаки, десятичную точку и знак е с показателем степени:
print('99')
print('98.6')
print('-1.5')
print('1.0e4')
